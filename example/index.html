phpchart

Minimal SVG charts generated on the server, without the need for JavaScript

phpchart has three core objectives: speed, simplicity and portability.

Rather than concentrating on animations and interactivity, phpchart renders on the server, which means there is no JavaScript required to build beautiful, functional charts. 


<?php
$chart = new Chart();
$chart->type = Chart::TYPE_BAR;
$chart->title = "Browser market share (January 2015)";
$chart->subtitle = "Source Wikipedia";
$chart->series = [
	new Series(
		// TODO: Define syntax
	]);
];
$chart->render("#container");
